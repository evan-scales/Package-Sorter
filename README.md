# Package-Sorter 1.0
EXTREMLY simple version of UPS CLT Air Hub

# Compiling and Running
```
// Windows / Linux
g++ *.cpp && ./a.out

// Mac
g++ -std=c++2a *.cpp && ./a.out
```

# Commands
1. `Scan Packages`
2. `Scan new ULD`
3. `Scan LIB Packages`
4. `Send off a Plane`
5. `Add ULD to Plane`
6. `Remove Packages from ULD`
7. `Run Tests`
